## 一、产品概述

### 1.1 产品定位

一款运行在浏览器端的封面图生成工具，用户通过左侧配置面板实时调整封面样式，右侧即时预览效果，最终导出高质量图片。面向博客作者、公众号运营、开发者等内容创作者。

### 1.2 核心价值

- **零门槛**：无需设计软件，浏览器即用
- **实时反馈**：所有设置即改即见
- **高度自由**：背景 / 图标 / 多行文本 / 水印全部可控
- **一键导出**：支持多平台尺寸预设 + 自定义分辨率

### 3.3 标题设置（**重点新增**：多文本图层）

#### 3.3.1 设计思路

将「单个标题」升级为「**文本图层列表**」，每个图层独立配置，支持任意数量。

#### 3.3.2 图层列表面板

```
┌─────────────────────────────────────────┐
│  T 标题设置                              │
├─────────────────────────────────────────┤
│  [图层 1] 星空   夜景          ✏️ 🗑️ ☰  │
│  [图层 2] XGDH.CN              ✏️ 🗑️ ☰  │
│                                         │
│  [+ 添加文本图层]                        │
└─────────────────────────────────────────┘
```

- 点击图层行展开该图层的配置（折叠/展开）
- ☰ 拖拽手柄支持图层排序（决定 z-index / 渲染顺序）
- 🗑️ 删除图层（至少保留 1 个）
- 点击「+ 添加文本图层」新建，默认复制上一图层的样式

| 预设名称 | 宽 × 高 | 用途 |

#### 3.5.2 **新增：比例快捷按钮**（核心新功能）

在宽高输入框上方新增比例快捷选择区：

```
┌──────────────────────────────────────────────────┐
│  📐 比例                                          │
│  [16:9]  [4:3]  [1:1]  [3:4]  [9:16]  [自由]     │
├──────────────────────────────────────────────────┤
│  ↔ 宽度          ↕ 高度                          │
│  [  1920  ] − +  [  1080  ] − +                  │
│                                                  │
│  ℹ️ 选择比例后，修改任意一边时另一边自动联动      │
└──────────────────────────────────────────────────┘
```

**交互逻辑：**
- 点击比例按钮（如 `16:9`），当前比例高亮，宽高值按当前宽度自动计算高度
- 选中「自由」后宽高独立输入，互不联动
- 手动输入宽高时，若已选中某比例，另一边联动更新；「自由」时不联动
- 比例按钮以胶囊样式展示，选中态绿色高亮

| 行为 | 说明 |
|------|------|
| 预览比例 | 始终按导出比例等比缩放展示，填满预览区 |

## 六、技术选型建议

| 层面 | 建议 |
|------|------|
| 框架 | Vue 3 + Vite + TypeScript |
| 状态管理 | Pinia（多图层配置需要统一 store） |

| 字体加载 | FontFace API 动态加载，确保导出时字体已渲染 |
| 文件下载 | `FileSaver.js` 或原生 `a[download]` |
| 颜色选择器 | `vue-color-kit` 或 `Coloris` |
| 拖拽排序 | `@vueuse/core` 的 `useSortable` 或 `Sortable.js` |
| 持久化 | localStorage（配置 JSON 序列化） |

## 八、页面布局规范

```
┌──────────────────────────────────────────────────────────┐
│  🖼 实时预览        [主题选择] [保存配置] [重置配置]      │
├────────────────────┬─────────────────────────────────────┤
│  [背景][图标][标题]│                                     │
│  [水印][导出]      │         实时预览区                  │

│  └──────────────┘  │                                     │
└────────────────────┴─────────────────────────────────────┘
```

- 左侧配置区固定宽度约 420px，右侧预览区自适应剩余空间
- 移动端折叠为上下布局：预览在上，配置在下（可折叠）
- 配置区内各 Tab 内容区域高度可滚动

## 九、交互细节说明

### 多文本图层

1. 新建图层时自动命名为「图层 N」，用户可重命名
2. 图层列表默认展开第一个，其余折叠
3. 删除图层前弹出确认：「删除后不可恢复，确认删除「图层 2」？」
4. 图层数量上限：10 个（超出后「添加」按钮置灰并 tooltip 提示）

### 比例联动

1. 切换比例按钮 → 以当前宽度为基准重算高度
2. 锁定比例后修改宽度 → 高度联动；修改高度 → 宽度联动
3. 切换导出平台预设 → 自动识别对应比例并高亮

### 导出流程

1. 点击「导出图片」
2. 按配置分辨率渲染画布（等待字体加载完成）
3. 转为目标格式 Blob
4. 触发浏览器下载
